=encoding utf-8

=head1 NAME

Unicode::GCString::JA_JP - UAX #29 書記素クラスタの列としての文字列

=head1 SYNOPSIS

    use Unicode::GCString;
    $gcstring = Unicode::GCString->new($string);
    
=head1 DESCRIPTION

B<注意: このモジュールはアルファバージョンなので、バグや不確定な仕様をいくつも含んでいる。>

Unicode::GCString はUnicode文字列を、Unicode標準附属書29 [UAX #29] で定義される「拡張書記素クラスタ」〔extended grapheme cluster〕の列として扱う。

B<書記素クラスタ>〔grapheme cluster〕は、Unicode文字の列で、ひとつのB<書記素基底>〔grapheme base〕と、付加的なB<書記素エキステンダ>〔grapheme extender〕および/またはB<前置文字>〔prepend character〕から成る。これは人が「文字」とみなすものに近い。

=head2 公開インタフェース

=head3 コンストラクタ

=over 4

=item new (STRING, [LINEBREAK])

I<コンストラクタ>。
Unicode文字列 STRING から新たに書記素クラスタ文字列
(Unicode::GCString オブジェクト) を作る。
オプションの L<Unicode::LineBreak> オブジェクト LINEBREAK は分割の仕様を制御する。

=item copy

I<コピーコンストラクタ>。
書記素クラスタ文字列の複製を作る。

=back

=head3 文字列としての操作

=over 4

=item as_string

=item C<">OBJECTC<">

I<インスタンスメソッド>。
書記素クラスタ文字列をUnicode文字列に変換する。

=item columns

I<インスタンスメソッド>。
組み込みの文字データベースで決定される書記素クラスタ文字列の桁数を返す。
詳しくは L<Unicode::LineBreak::JA_JP/DESCRIPTION> を参照。

=item concat (STRING)

=item STRING C<.> STRING

=item STRING C<.=> STRING

I<インスタンスメソッド>。
書記素クラスタ文字列を結合する。
STRING のどちらかがUnicode文字列でもよい。

=item length

I<インスタンスメソッド>。
書記素クラスタ文字列に含まれる書記素クラスタの数を返す。

=item substr (INDEX, [LEN])

I<インスタンスメソッド>。
B<未実装>.
書記素クラスタ文字列の部分文字列を返す。

=back

=head3 書記素クラスタの列としての操作

=over 4

=item as_array

=item C<@{>OBJECTC<}>

=item as_arrayref

I<インスタンスメソッド>。
書記素クラスタ文字列を、書記素クラスタの情報の配列に変換する。

=begin comment

=item eot

I<インスタンスメソッド>。
B<未実装>。
現在の位置が書記素クラスタ文字列の最後かどうか調べる。

=end comment

=begin comment

=item flag (INDEX, [VALUE])

I<Undocumented>.

=end comment

=item item (INDEX)

I<インスタンスメソッド>。
INDEX番めの書記素クラスタの情報を配列参照として返す。

=begin comment

=item next

I<インスタンスメソッド>、反復的。
B<未実装>。
次の書記素クラスタの情報を配列参照として返す。

=item prev

I<インスタンスメソッド>。
B<未実装>。
書記素クラスタ文字列中の位置をひとつ戻す。

=item reset

I<インスタンスメソッド>。
B<未実装>。
書記素クラスタ文字列の次の位置を初期化する。

=item rest

I<インスタンスメソッド>。
B<未実装>。
書記素クラスタの残りを返す。

=end comment

=back
